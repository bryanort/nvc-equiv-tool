🎓 Confidence level guide

  • High  Same WECM content. Apply with standard dept sign-off. E.g. ITSY 1300 ↔ ITSY 1342.
  • Medium  Substantial overlap. Requires faculty outcome comparison. E.g. ITSY 2301 ↔ ITCC 2343.
  • Low  Related topic. Pathway counseling only. E.g. AWS sequence.
